# Jonah and Noel's Asteroid Assault Game

GitHub Pages-ready build.

## Upload location

Place this entire folder inside your repository as:

```text
evelyn-games/
└── games/
    └── asteroid-assault/
        ├── index.html
        ├── assets/
        └── README.md
```

The game URL will be:

```text
https://yijinnchen16.github.io/evelyn-games/games/asteroid-assault/
```

## Upload through GitHub

1. Open the `yijinnchen16/evelyn-games` repository.
2. Create `games/asteroid-assault/` if it does not exist.
3. Upload `index.html`, the complete `assets` folder, and this README.
4. Commit the files to the branch used by GitHub Pages, normally `main`.

## Controls

- Move: Arrow Up / Arrow Down or W / S
- Mobile: drag vertically
- Transform: Space or the on-screen mode button
